使用clang进行编译
启用运行时类型信息rtti
会出现需要C++1z作为标准的warning

该程序主要为使用函数模板推断的方式进行编译时运算，
同时也展示了一些简单使用宏进行编译时计算的方法。
使用<ctime>中的clock()函数说明运行的时间为0，直观地证明计算是在编译时。

尽管总体的代码量并没有很大，但是在大量使用模板的过程中
每加任何一句代码时都可能引发大量编译错误，
且无法通过调试只能通过语法树进行错误的推断，
比正常实现一些程序所需要花费的时间和精力多得多。

大体实现了以下四部分：
1、简单使用宏进行计算圆面积、计算可变参数个数
2、结合fibonacci数列、阶乘数列和if_then_else进行简单运算
3、switch（包括正向(随机)和反向，其中正向利用函数模板特化为另一个类的模板的方式实现，反向利用可变长参数模板实现）
   进行简单运算
4、实现编译时字符串的连接
